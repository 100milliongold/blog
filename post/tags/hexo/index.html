hexo password 설정

2019년 7월 31일 · 약 1분
  1. hexo-blog-encrypt 모듈 설치
    $ npm install --save hexo-blog-encrypt
  2. _config.yml 에 다음항목 추가
    encrypt:
        enable: true
  3. post 상단에 다음 항목추가
    ---
    password: mikemessi
    abstract: Welcome to my blog, enter password to read.
    message: Welcome to my blog, enter password to read.
    ---
